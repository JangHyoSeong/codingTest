## 파일과 커밋 형식
- 문제 풀이는 각 폴더에 README.md 파일과 소스코드로 이루어져 있음.
- 폴더의 이름은 문제번호_문제제목
  - 파일의 이름은 문제번호_문제제목.md
  - sourceCode.py
- README.md 파일의 형식은 form.md를 따름.
- 커밋 메세지는 다음과 같음

`git commit -m "[사이트] 문제번호/제목 사용언어 기타필요한메세지"`

```bash
# 커밋 메세지 예시
[baekjoon] 1001 py
[programmers] lv1/더하기 py
```
